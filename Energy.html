Determining Top Geospatial Features for Electricity Demand Prediction¶

Problem Statement¶

The Project aims to develop a machine learning algorithm that can determine the Top Geospatial Features available on the dataset that can predict Electricity Demand in Luzon, Philippines. Specifically, we would like to answer the following problems:

  1. Can Electricity Demand in Luzon be predicted using only Geospatial features?
  2. How important are these features in forecasting energy demand?

Using this project, we aim to create a proof of concept that geospatial features seen in the Luzon are possible avenue for insights in deciding future energy demand. By doing so, we are opening a new avenue of further research in using geospatial analysis in conjecture with electrical engineering and at the same time also use the model as a means for creating new horizons for new research in electricity forecasting.

Datasets to be Used¶

  1. Open Street Maps (OSM)

Using the OSM Database, we mined the following respective data:

a. Geospatial Boundaries of each City/ Municipality in Luzon
b. Administrative Data of Each City/ Municipality in Luzon

    1. Amenities Count per Municipality
    2. Tourism Establishment Count per Municipality
    3. Leisure Establishment Count per Municipality
    4. Historic Establishment Count Per Municipality
    5. Highway Data Count Per Municipality

c. Substation Locations and Name in Luzon
  1. Independent Market Operator of the WESM (IEMOP) Energy Demand Data

Through the IEMOP Database, we used the following items:

a. Hourly Electricity Demand (Megawatts) of Various Substations within the Luzon Grid
  1. Freemeteo.co.uk Weather Data

    df = pd.read_csv(x).drop(['Unnamed: 0','Unnamed: 0.1','Wind Gust','Date'],axis=1)
    df['City']= re.findall(r'weather/(.+?)_',x)[0]
    df['Time'] = df.Time.apply(timex)
    df['Temperature'] = df.Temperature.str.replace("°C",'')
    df['Relative Temperature'] = df['Relative Temperature'].str.replace("°C",'')
    df['Relative Humidity'] = df['Relative Humidity'].str.replace("%",'')
    df['Dew Point'] = df['Dew Point'].str.replace("°C",'')

    df['Wind Speed'] = df['Wind Speed'].apply(findspeed)
    df['clouds'] = df['Condition'].apply(clouds)
    df['clods_height'] = df['Condition'].apply(clouds_h)
    df = df.drop(['Condition'],axis=1)
